# Go 程序设计语言

日期：2026年2月

主题：学习《The Go Programming Language》的读书笔记。



时隔一年再次学习 go 语言，我最先接触的书就是这一本了。我注册 github 账号时创建的第一个 repo 就是 bravo-go。写下了hello world，后来因为学业的关系就中断学习了。那段时间（包括之前）写下的代码应该也有上百行了吧，但是我没保存。

| 其他编程语言的掌握情况 |                                                              |
| ---------------------- | ------------------------------------------------------------ |
| C++                    | 上课教的，几乎没学会（我不会写）                             |
| Python                 | 知道什么是面向对象了，也知道MVC，会写简单的网页              |
| Ruby                   | 好玩，但是我为啥不用 Wordpress 搭建个人网站呢                |
| Java                   | 知道什么是 SpringBoot 全家桶，Jwt，CORS，测试，数据库，还有一堆点点点 |
| Javascript             | 知道什么是前端框架（Angular模板，React 钩子函数），其他的一知半解 |

可以看出我在一开始学习编程的时候，是比较随便的。

除了敲代码，我在上大二的时候还接触过虚拟机（VmWare）。使用过时的远程连接软件（Xshell） 敲 Linux 命令。

那真是一段令人头疼的经历。

下面这些经历也发生在大二时期：我喜欢在Bilibil上看网络工程和网络安全的视频（路飞学城的录课，和软考培训视频），我加入过一个网络安全小组的QQ群，还有各种打培训班广告的微信群（那些都是割韭菜的，我那个时候不懂，交了40块钱买课），还有无数次往返图书馆，坐了一天都不知道在看什么，所以没有印象。

2023年，AI的问世将我们带进了人工真能的世界，人工智能可以是阿尔法狗，也可以成为写作论文的工具，一个能陪你聊天的助手。

我是个电脑小白，但是学习这些东西，至少能让事情变得简单一些，这难道是我学习编程的理由吗？

反正我知道该怎么保存我写的东西就行了……
